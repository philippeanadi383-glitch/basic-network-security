# Basic Network Security — ACL

Simulation d'un réseau sécurisé avec Cisco Packet Tracer.

## Topologie
- 2 PCs sur des réseaux différents
- 1 Routeur ISR4331

## Sécurité implementée
- ACL étendue (100) bloquant le trafic entre 192.168.1.0/24 et 192.168.2.0/24

## Test
- Ping de PC1 vers PC2 → 100% loss ✅
